
This program simulates a playlist by letting the user add/delete songs, find a certain song in the playlist, and 
show various attributes of a playlist like category or size. This was coded and compiled in an Unix environment


How to compile the program:

Locate the directory with all the program's files and do the following command to compile;
g++ menu.cpp playlist.cpp song.cpp



This should of made an a.out executable. Run the executable to run the simulation
